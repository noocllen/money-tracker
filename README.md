💰 Money Tracker

Money Tracker — это приложение для учета личных финансов, разработанное с использованием FastAPI. Оно позволяет управлять кошельками, учитывать доходы и расходы, выполнять переводы между счетами и отслеживать историю финансовых операций.

Возможности
Регистрация и авторизация пользователей
Создание и управление кошельками
Поддержка нескольких валют
Добавление доходов и расходов
Переводы между кошельками
Просмотр истории операций
REST API с документацией Swagger
Технологии
Python
FastAPI
SQLAlchemy
PostgreSQL
Pydantic
